ES6新增Set和Map的简单使用

1. Set
基本用法
  • Set类似于数组，但成员的值都是唯一的，没有重复的值；
  • Set 是一个构造函数；

Set函数可接收一个数组用来初始化；

Set实例常用内置函数；

  • add 添加一个成员
  • delete 删除某个成员 返回true/false
  • has 判断内部是否有某个成员
  • clear 清空内部成员
  • .size 内置属性 获取成员个数 类似于Array.length;
2. Map
基本用法

Map 可接受一个数组作为参数，该数组的成员是一个个表示键值对的数组；

Map实例常用内置函数；

  • set 添加一个成员
  • get 获取某个键值对的value
  • delete 删除某个键值对 返回true/false
  • has 判断内部是否有某个属性
  • clear 清空内部成员
  • .size 内置属性 获取成员个数 类似于Array.length;

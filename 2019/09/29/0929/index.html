父组件向子组件传值

  1. 使用props 方式传值，父组件引用组件，使用组件后，可通过子组件内定义的 props 属性，
    使用该属性名传值到子组件内;
  2. props 可以传入String,Number,Object,Boolean,Array数据类型;
  3. prop 验证：设置required 参数 可选择是否当前属性必传;
  4. default 设置如若不传将展示的内容;
  5. props 传值为单向数据流;
    注意:子组件内不能直接修改父组件传来的props；、;
